# 🧠 Career Assessment System

A comprehensive psychological testing platform built with React, featuring 8 different career and personality assessment tests in Gujarati language.

## ✨ Features

### 🧪 **8 Comprehensive Tests**
- **MBTI Personality Test** - 80 questions for personality type assessment
- **Big Five Personality Test** - 40 questions for core personality dimensions
- **Multiple Intelligences Test** - 72 questions for intelligence type assessment
- **RIASEC Career Test** - 60 questions for career interest assessment
- **SVS Value Survey** - 50 questions for personal values assessment
- **Decision-Making Style Test** - 25 questions for decision style assessment
- **VARK Learning Style Test** - 20 questions for learning preference assessment
- **Life Situation Assessment** - 25 questions for life circumstances evaluation

### 🌟 **Key Features**
- **Gujarati Language Support** - Full Gujarati interface for Indian students
- **Professional Results Display** - Detailed analysis with charts and career recommendations
- **Responsive Design** - Works perfectly on mobile, tablet, and desktop
- **Interactive UI** - Smooth animations and modern user experience
- **Results Download** - Save test results as text files
- **Progress Tracking** - Real-time progress monitoring during tests

### 🎨 **UI/UX Features**
- Modern gradient design with Tailwind CSS
- Smooth animations using Framer Motion
- Interactive charts and visualizations
- Professional color scheme and typography
- Mobile-first responsive design

## 🚀 Getting Started

### Prerequisites
- Node.js (version 14 or higher)
- npm or yarn package manager

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/career-assessment-system.git
   cd career-assessment-system
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm start
   ```

4. **Open your browser**
   Navigate to `http://localhost:3000`

### Build for Production

```bash
npm run build
```

## 🧪 Test Details

### MBTI Personality Test
- **Questions**: 80 (20 per dimension)
- **Dimensions**: E/I, S/N, T/F, J/P
- **Duration**: 45-60 minutes
- **Results**: Complete personality type with career recommendations

### Big Five Personality Test
- **Questions**: 40 (8 per dimension)
- **Dimensions**: Openness, Conscientiousness, Extraversion, Agreeableness, Neuroticism
- **Duration**: 25-35 minutes
- **Results**: Personality profile with career suggestions

### Multiple Intelligences Test
- **Questions**: 72 (8 per intelligence type)
- **Types**: Musical, Logical, Intrapersonal, Bodily, Naturalistic, Visual, Linguistic, Interpersonal, Existential
- **Duration**: 45-60 minutes
- **Results**: Intelligence profile with learning recommendations

### RIASEC Career Test
- **Questions**: 60 (10 per interest area)
- **Areas**: Realistic, Investigative, Artistic, Social, Enterprising, Conventional
- **Duration**: 30-40 minutes
- **Results**: Career interest profile with job recommendations
